# Security Policy

## Supported Versions

`@pkg-nec/http-server` supported versions:

| Version | Supported |
|---------|-----------|
| 14.1.x  | ✔️ Yes    |

Upstream `http-server` (`http-party/http-server`) is out of scope for this
policy. If you're using the unscoped `http-server` package, please report
to that project instead.

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub's
[private security advisory](https://github.com/pkg-nec/http-server/security/advisories/new)
form on `pkg-nec/http-server`. We aim to acknowledge reports within 5
business days.

Public issues are also acceptable for lower-severity findings where you
believe a public discussion will not put users at increased risk.
